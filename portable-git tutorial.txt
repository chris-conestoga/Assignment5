1) launch git portable
2) select clone existing repository
3) get the http key for the repository from github
4) set that as the source
5) set a new folder as the destination
6) go to edit -> options -> enter your username and email

if you need to commit new changes:
1) commit -> rescan (new changes will appear under “unstaged changes”
2) commit -> stage to commit
3) enter commit description
4) click “commit” button beside description
5) click “push” button below

if you need to update your files from the repository:
1) remote -> fetch from -> origin
2) say ok or whatever to the dialogue
3) merge -> local merge
4) tracking branch should be selected, should show “origin” in the listbox
5) say ok or whatever to the dialogue

if anything goes wrong at any time:
1) you’re fucked
